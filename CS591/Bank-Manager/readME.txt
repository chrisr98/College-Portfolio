Make sure all the *.java files and *.txt files are in the same folder.
To run the version presented today just type open the terminal-> go to dir with the files
type -> "javac Main.java && java Main"

If you want to run the methods that we didn't have time to demo during presentation
Uncomment the main function all the way at the bottom of Bank.java.
While in the same dir type -> "javac Bank.java && java Bank"
You can confirm the changes in the account and transaction database txt files
